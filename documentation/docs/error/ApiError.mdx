# ApiError

Class for getting API error message from parameter in `catch` block.

:::note
Class can be used in frontend and backend
:::

## Import

<Tabs>
  <TabItem value="ES5" label="ES5" default>
    
    ```jsx

    ```

  </TabItem>
  <TabItem value="ES6" label="ES6">

     ```jsx

    ```

  </TabItem>
</Tabs>

## Example

```jsx live
function Example() {
  const [statusCode, setStatusCode] = useState(0);
  const [error, setError] = useState("");

  async function getApiData() {
    let error = null;

    try {
      await axios({
        url: "http://localhost:8080",
        method: "get",
      });
    } catch (err) {
      console.error(err);

      error = new ApiError(DefaultError.message(err), 500);
      if (ApiError.isApiError(err)) {
        error = new ApiError(ApiError.default(err), err.response.status);
      }
    }

    setStatusCode(error.status);
    setError(error ? error.message : "");
  }

  getApiData();

  return (
    <>
      <b>Status:</b> {statusCode}
      <br />
      <b>Message:</b> {error}
    </>
  );
}
```

## Reference

### Create Object

`new ApiError(message, status)`

| Name      | Type     | Default Value | Description                   |
| --------- | -------- | ------------- | ----------------------------- |
| `message` | `string` |               | error message                 |
| `status`  | `number` | 500           | status code from API response |

### Object Properties

:::note
Includes all properties from [`DefaultError`](/docs/error/DefaultError) class
:::

| Name     | Type     | Static | Default Value |
| -------- | -------- | ------ | ------------- |
| `status` | `number` | yes    | 500           |

### Methods

:::note
Includes all methods from [`DefaultError`](/docs/error/DefaultError) class
:::

#### `static isApiError(err)`

##### Parameters

- <b>Type:</b> `Error`

- <b>Description:</b> `Error` parameter from `catch` block

##### Returns

- <b>Type:</b> `boolean`

- <b>Description:</b> `true` if error is from sequelize; otherwise, `false`

#### `static default(err)`

##### Parameters

- <b>Type:</b> `Error`

- <b>Description:</b> `Error` parameter from `catch` block

##### Returns

- <b>Type:</b> `string`

- <b>Description:</b> message from `Error` parameter from `catch` block
