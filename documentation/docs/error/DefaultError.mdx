# DefaultError

Class for getting error message from parameter in `catch` block.

:::note
Class can be used in frontend and backend
:::

## Import

<Tabs>
  <TabItem value="ES5" label="ES5" default>
    
    ```jsx
    const { DefaultError } = require("js-ts-kit");
    ```

  </TabItem>
  <TabItem value="ES6" label="ES6">

     ```jsx
    import { DefaultError } from "js-ts-kit";
    ```

  </TabItem>
</Tabs>

## Examples

```jsx live
function Example() {
  let error = "";

  try {
    throw new Error("This is a test error message");
  } catch (err) {
    console.error(err);

    error = DefaultError.message(err);
  }

  return error;
}
```

```jsx live
function JsonExample() {
  const errorJson = {
    status: 400,
    error: {
      message: "Unable to publish",
      payload: {
        nested: {
          exceptionMessage: "Invalid payload",
        },
      },
    },
  };

  return DefaultError.json(errorJson);
}
```

### Methods

#### `static message(err)`

##### Parameters

- <b>Type:</b> `Error`

- <b>Description:</b> `Error` parameter from `catch` block

##### Returns

- <b>Type:</b> `string`

- <b>Description:</b> message from `Error` parameter from `catch` block

#### `static json(err)`

##### Parameters

- <b>Type:</b> `Error | any`

- <b>Description:</b> JSON-like error value whose nested message fields should
  be traversed

##### Returns

- <b>Type:</b> `string`

- <b>Description:</b> extracted error message string from nested JSON objects,
  arrays, and error instances
